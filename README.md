# Bitwise Tests!

## Setup

### Install Node
Navigate your browsing device here: http://nodejs.org/

### Clone the repo
```
git clone git@github.com:itsthatguy/bitwise-tests.git
```

### Install project dependencies
```
npm install
```

### Run the tests
```
npm test
```
